type existential_restriction =
| At_toplevel

no existential types at the toplevel

| In_group

nor with let ... and ...

| In_rec

or recursive definition

| With_attributes

or let[@any_attribute] = ...

| In_class_args

or in class arguments class c (...) = ...

| In_class_def

or in class c = let ... in ...

| In_self_pattern

or in self pattern
